# 👑 Giannis Antetokounmpo — Greek Freak

Um projeto inspirado em Awwwards que retrata a trajetória de superação e domínio de Giannis Antetokounmpo.

## ✨ Visão Geral

O site mostra a evolução de Giannis desde suas origens até se tornar um dos maiores jogadores da NBA.

## 🚀 Funcionalidades

* Storytelling com scroll (linha do tempo interativa)
* Animações suaves e transições elegantes
* Destaque para conquistas e momentos marcantes
* Design responsivo

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* GSAP

## 🎨 Conceito de Design

* Cores: verde, branco e dourado
* Visual limpo com foco em narrativa
* Tipografia moderna e elegante

## 📂 Estrutura do Projeto

```
/assets
index.html
```

## 📸 Objetivo

Contar a história inspiradora de Giannis, destacando sua jornada até o topo da NBA.
